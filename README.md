# 《180秒 · 生命赌局》

一个用"上瘾机制反向揭露赌博真相"的网页小游戏。把**时间当作生命与筹码**：你只有 180 秒，不断和庄家对赌，越赢越停不下来——直到输光，游戏揭示这一切从一开始就是被精心设计的骗局。

> 本游戏旨在警示：**远离赌博**。赌局里唯一的胜利，就是不再下注。

## 玩法

- 起始生命 **180 秒**，倒计时实时流逝（犹豫也在掉血）。
- 每局下注若干秒：**赢则双倍返还，输则永久失去，平局退还**。
- 赢了生命可以**突破 180 秒上限**。
- 下注额可选预设筹码、自定义秒数，或**动态全押**（额度实时等于当前生命）。

## 游戏阵容（随局数由简到繁、由低风险到高风险解锁）

| 解锁局数 | 游戏 | 特点 |
|---|---|---|
| 第 0 局 | 摇骰子·猜大小 | 最简单，蜜月期 |
| 第 4 局 | 扑克比大小 | 比点数 |
| 第 8 局 | 百家乐 | 押庄/闲/和 |
| 第 12 局 | 21 点 | 要牌/停牌，需决策 |
| 第 18 局 | 炸金花 | 比牌型，烧脑 |
| 第 25 局 | 梭哈 | 五张比牌，最复杂 |
| 第 32 局 | 老虎机 ⚠ | 高风险，"差一点"机制 |
| 第 40 局 | 俄罗斯轮盘 💀 | 赌命：中弹归零，幸存×4 |

## 暗藏的"上瘾设计"（也是警示的核心）

- 诱饵开局（前几把故意让你赢）
- 隐形赔率衰减（界面永远显示公平 50/50，真实胜率随局数偷偷下滑）
- 赢家诅咒、连败救援、差一点动画、沉没成本提示、临终稻草……

## 两个结局

- **死亡揭示**：输光后亮出真实胜率、时间统计，点题"庄家永远是赢家，这不是运气，是数学"。
- **真正的胜利**：主动点"离开"按钮带着剩余生命走开——"能转身走开的人，才是赢家"。

## 技术

纯前端，零依赖，零构建：

- `index.html` — 页面结构
- `style.css` — 赌场风格视觉与动画
- `engine.js` — 核心引擎（倒计时 / 隐藏赔率 / 上瘾机制 / 结局）
- `games.js` — 8 个赌局模块
- `sound.js` — 用 Web Audio API 实时合成的音效与背景音乐
- `assets/casino.png` — 赌场场景背景图

## 运行

直接用浏览器打开 `index.html` 即可（记得先点「坐上赌桌」，浏览器要求用户手势后才能播放声音）。
